## 第二步 下载c/c++编译器
### 这里我使用MinGW
> 实际上 MinGW 并不是一个 C/C++ 编译器，而是一套 GNU 工具集合。除开 GCC (GNU 编译器集合) 以外，MinGW 还包含有一些其他的 GNU 程序开发工具 (比如 gawk bison 等等)。

### 找到[资源](https://winlibs.com/)下载压缩包
![](/img/blog/blog_date_to_replace/04.png)
